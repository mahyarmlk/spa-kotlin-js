# Kotlin Single Page App

Single page application example app in kotlin

## Demo
https://techprd.github.io/spa-kotlin-js/#/home/

### dependencies:

https://github.com/techprd/kotlin-material

### clean build project

```bash

./gradlew clean build

```
### run dev server

```bash

./gradlew run

```

### stop dev server


```bash

./gradlew stop

```
